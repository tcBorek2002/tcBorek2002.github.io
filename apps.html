Proximity sensor test

Proximity sensor test is an Android app that allows the user to test the proximity sensor on their phone. It has two types of tests. Basic test: check if the proximity sensor near or far value is reached by looking at the colour of the screen. Distance test: display the exact distance value the proximity sensor senses. Finally, there is also the sensor info page that will display information available about the proximity sensor.

Previously, I have published an older version of proximity sensor test that is not available any more. The current proximity sensor test has over 200 thousand downloads in the Google Play Store. It is one of the most used apps for checking the proximity sensor.
